# ServerBot-Base
Minimal version of ServerBot project

ServerBot-Base uses minimized version of [ServerBot](https://github.com/kamile320/serverbot) project.  
It has only few commands to show schemes for '.' and '/' -type commands. Also uses less python libraries.

New releases will be uploaded after upload of the last SB release (v1.X.X), eg. v1.9.3 -> last before v1.10

To learn how install and use bot, read ServerBot Manual [[PL]](https://kamile320.github.io/ServerBot/manualPL.html) [[EN]](https://kamile320.github.io/ServerBot/manualEN.html)
